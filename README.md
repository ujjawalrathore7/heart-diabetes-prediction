# Design of Explainable AI Heart Disease and Diabetes Prediction

## Project Overview

This project presents an **Explainable Artificial Intelligence (XAI)** based healthcare prediction system for detecting **Heart Disease** and **Diabetes** using machine learning and deep learning techniques. The system analyzes medical and clinical parameters to provide accurate disease prediction along with interpretable explanations using **SHAP (SHapley Additive Explanations)**.

The main objective of this project is to assist in **early disease detection**, improve healthcare decision-making, and provide transparent AI predictions for better trust and reliability in medical applications.
